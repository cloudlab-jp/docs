# CloudLab JP

CloudLab JP is an open-source initiative focused on **Cloud Engineering**, **Platform Engineering**, **DevOps**, **Infrastructure as Code**, and modern software delivery.

Our goal is to build production-ready learning resources, reusable templates, automation, and engineering best practices that anyone can use.
